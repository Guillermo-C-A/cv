Python

Estudio sobre el coste de seguros médicos

Tipo: Regresión lineal y ANOVA

Descripción del proyecto

Es este proyecto se juega con una base de datos cuya finalidad es la de estudiar el coste final de un seguro médico en función de varios parametros como pueden ser: número de hijos, si la persona es fumadora, el índice de masa corporal, etc... La base de datos se divide en dos grupos: la base de datos de entrenamiento (70%) y la de test (30%).

El estudio se divide en dos bloques. El primer bloque es un analisis descriptivo de la base de datos simple junto con una tabla ANOVA y la segunda parte consta del modelo de regresion lineal.

Podras encontrar este proyecto en este enlace.

Estudio sobre el coste de seguros médicos

Tipo: Regresión lineal y ANOVA

Descripción del proyecto

Es este proyecto se juega con una base de datos cuya finalidad es la de estudiar el coste final de un seguro médico en función de varios parámetros como pueden ser: número de hijos, si la persona es fumadora, el índice de masa corporal, etc... La base de datos se divide en dos grupos: entrenamiento (70%) y test (30%).

El estudio se divide en dos bloques. El primer bloque es un análisis descriptivo de la base de datos simple junto con una tabla ANOVA y la segunda parte consta del modelo de regresión lineal.

Podrás encontrar este proyecto en este enlace.
